# Navbar Component

Top navigation bar for the application.

## Props

| Prop           | Type     | Default  | Description                                |
| -------------- | -------- | -------- | ------------------------------------------ |
| `viewMode`     | string   | -        | Current view mode ('LIVE' or 'HISTORICAL') |
| `setViewMode`  | function | -        | Handler to change view mode                |
| `systemStatus` | string   | 'ONLINE' | Current system status displayed in header  |

## Features

- Displays Application Title
- View Mode Switcher
- System Status Indicator
- Integrated Theme Toggle
